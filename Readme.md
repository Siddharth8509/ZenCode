# ZenCode - Platform Documentation

## 🚀 Overview
ZenCode is a comprehensive, AI-powered platform designed to provide a structured roadmap to technical interview success. It empowers software engineering candidates by offering curated Data Structures and Algorithms (DSA) practice, real-time code compilation, AI-driven mock interviews, and automated resume building. 

Whether you are a beginner looking to build problem-solving intuition or an experienced developer aiming to crack top-tier tech interviews, ZenCode systematically guides your preparation.

---

## 💻 Day-to-Day Use Cases
ZenCode is built to be a daily driver for interview preparation. Here is how a user might interact with the platform on a day-to-day basis:

1. **Daily Problem Solving (The Core Loop):**
   * **Log in daily** to view the "Today's Sprint" tracker on the dashboard.
   * **Navigate to the Problemset** to find curated DSA challenges organized by topic (e.g., Arrays, Trees, Dynamic Programming).
   * **Use the built-in IDE** (powered by Monaco Editor) to write code and execute it in real-time against test cases.
   * **Track progress:** Completed problems update the user's progress bar, maintaining a persistent run streak to build consistency.

2. **AI Mock Interviews (Weekly Practice):**
   * Before a real interview, users can launch the **AI Mock Interviewer**.
   * Practice behavioral and technical questions dynamically generated by AI.
   * Receive real-time, constructive feedback on answers, helping refine communication and technical articulation under pressure.

3. **Resume Optimization (Pre-Application):**
   * Use the **Smart Resume Maker** to auto-generate an ATS-friendly resume.
   * The tool perfectly highlights coding skills, core competencies, and statistics directly pulled from the user's ZenCode solving history.

4. **Preliminary Round Prep:**
   * Sharpen quantitative math and logical reasoning skills using the **Adaptive Aptitude** quiz module, tailored through AI to match the user's skill level.

## ✨ Core Functionalities

### Frontend (Client-Side)
* **Interactive Code Execution Environment:** A full-fledged code editor (Monaco Editor) integrated directly into the browser, complete with syntax highlighting and a split-pane layout to view problems and run code side-by-side.
* **Real-time User Dashboard:** Displays solving statistics, daily streaks, unlocked topics, and overall progress metrics.
* **AI Pro Tools Suite:** Provides a seamless interface to interact with the Mock Interviewer, Resume Maker, and Aptitude quizzes, featuring premium UI animations (Framer Motion).
* **Markdown Rendering:** Beautifully formats complex DSA problem descriptions (using `react-markdown` and `@uiw/react-markdown-preview`).
* **Admin Panel Context:** Allows administrators to manage problem sets, update test cases, and monitor user statuses in real-time.

### Backend (Server-Side)
* **Secure Authentication & Authorization:** Implements robust JWT (JSON Web Tokens) based authentication with secure, HTTP-only cookie handling. Includes route protection middleware and role-based access control (Admin vs. User).
* **Code Execution Engine Integration:** Securely receives code submissions from the frontend, compiles/runs them against defined test cases in an isolated environment, and returns detailed execution results (stdout, stderr, execution time).
* **AI Integration (Google Gemini):** Utilizes the Google Gemini SDK (`@google/genai`) to power the platform's advanced features, such as generating mock interview questions, evaluating answers, and dynamically adjusting aptitude test difficulty.
* **Data Persistence:** Manages user profiles, problem lists, test cases, and historical submission statuses using MongoDB.
* **Caching Layer:** Utilizes Redis to cache frequently accessed data (like the main problemset list) to optimize database load and dramatically improve frontend response times.

## 🛠️ Technology Stack

### Frontend
- **Framework:** React 19 / Vite
- **Routing:** React Router DOM v7
- **State Management:** Redux Toolkit (`@reduxjs/toolkit`)
- **Styling & UI:** 
  - Tailwind CSS v4 (with custom `@theme` configuration)
  - HeroUI Component Library
  - Framer Motion & GSAP (for scroll animations and page transitions)
  - Lenis (for smooth scrolling)
- **Editor & Content:**
  - Monaco Editor (`@monaco-editor/react`)
  - React Split Pane (`react-resizable-panels`)
  - React Hook Form & Zod (form validation)

### Backend
- **Runtime:** Node.js (v22.x)
- **Framework:** Express.js 5
- **Database:** MongoDB (via Mongoose ODM)
- **Caching:** Redis (`redis` client)
- **Authentication & Security:** 
  - JWT (`jsonwebtoken`)
  - bcrypt (for secure password hashing)
  - CORS and Cookie Parser
- **AI Integration:** Google Gemini SDK (`@google/genai`)
- **Deployment Environments:** 
  - Configured for serverless/edge deployment (e.g., Vercel, Render) with considerations for CORS and stateless execution.
